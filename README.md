# Mundo de Léo e Lucas — ABC, Expedições e Quebra-cabeças

Versão pronta para GitHub Pages, sem build obrigatório.

## Estrutura

- ABC do Lucas: 26 letras, cenas próprias, vozes e interações.
- Expedições do Léo: 10 missões investigativas com ferramentas, pistas e consequências.
- Quebra-cabeças: modo Lucas e modo Léo com dificuldades diferentes.
- Área dos responsáveis: progresso, diário, exportação/importação.
- PWA offline.

## Publicação

Envie o conteúdo deste ZIP diretamente para a raiz do repositório GitHub Pages.

O `index.html` deve ficar na raiz.

## Áudio

Ao abrir, toque em "Começar" para liberar sons e vozes.

## Observação

As vozes são sintéticas em português brasileiro, pré-geradas em WAV.
